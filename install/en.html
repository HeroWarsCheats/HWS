Hero Wars Simulator

Installing the Hero Wars Simulator
Download the extension
Go to the Google Store and download the extension ( click here )

It's important to have Google Chrome!!
Go to the "extensions" section of Google Chrome
1 First click the 3 dots
2 Then click the "extensions" section
3 Finally click "manage extensions"
Make sure developer mode is enabled
If it's not active, press the switch.
If necessary, restart your browser to validate the activation
Go to "details"
in the Hero Wars Simulator extension
"Allow User Scripts"
this item must be selected
You should see tables
in the top left when you open the game
Be sure to disable
any extensions that might cause a conflict
